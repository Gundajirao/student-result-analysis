#  Student Result Analyzing System

##  Description
The Student Result Analyzing System is a web-based application that allows users to store, manage, and view student academic results. It helps in calculating student performance and displaying results efficiently through a simple interface.

---

##  Features
- Add student details and marks  
- View student results  
- Calculate total marks and percentage  
- Display grades based on performance  
- Simple and user-friendly interface  

---

##  Tech Stack
- Frontend: HTML, CSS, JavaScript  
- Backend: Python (Django / Flask)  
- Database: SQLite
